# SimpleOpenGL 2D Robot Animation

A simple C/OpenGL project demonstrating 2D planar robot animation using GLFW and GLAD.

## Features

- Animates a 2-link planar robot arm.

## Work in progress
- Inverse kinematics
- Path planning
- Inertial considerations

### Build

```bash
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
cmake --build .
